# AGENTS

This repository supports human and AI collaboration. Please follow these guidelines when working here:

## General

- Keep changes focused and minimal.
- Prefer readability over cleverness.
- Follow existing project conventions and formatting.
- Update or add tests for behavior changes when applicable.

## Workflow

- Inspect relevant files before editing.
- Explain what you changed and why.
- Avoid unrelated refactors.

## Flutter/Dart

- Keep imports tidy and remove unused ones.
- Ensure platform-specific configurations (Android/iOS) remain consistent after edits.

## Documentation

- Update docs when behavior or configuration changes.
